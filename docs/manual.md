# 英文ラボ（Eibun-Lab） — 操作マニュアル

## 初回セットアップ

### 1. Gemini API キーの取得

1. [Google AI Studio](https://aistudio.google.com/) にアクセスしてログイン
2. 「Get API key」からAPIキーを発行
3. 発行したキーをコピー

### 2. アプリへの API キー登録

1. アプリ下部のナビゲーションから **設定** タブを開く
2. 「Gemini API キー」欄にコピーしたキーを貼り付ける
3. **保存** ボタンをタップ

> API キーはブラウザの LocalStorage にのみ保存されます。外部サーバーへは送信されません。

### 3.（任意）Google アカウントでログインしてクラウド同期する

複数の端末・ブラウザで添削履歴を共有したい場合は、設定ページの **アカウント** 欄から Google アカウントでログインできます。ログインすると、以後の添削履歴が自動的に Firebase 経由でクラウドと双方向同期され、別の端末でログインすれば同じ履歴を引き継げます。ログインしなくてもアプリの全機能は利用できます。

## 各ページの操作

### 練習（英文添削）

英作文を入力して Gemini AI に添削してもらうメインページ。

1. テキストエリアに英文を入力する
2. **添削する** ボタンをタップ
3. 添削結果が下に表示される

**表示内容**（以下はすべて常時表示されます。個別のON/OFF設定はありません）

| 項目               | 説明                                                   |
| ------------------ | ------------------------------------------------------ |
| 添削結果           | 文法・語法ミスの指摘と修正理由（Markdown 形式）        |
| 添削後の全文       | 修正を反映した完成版の英文                             |
| ミス一覧           | カテゴリ・元の表現・正しい表現・解説                   |
| 自然な表現の提案   | ネイティブらしい言い回し                               |
| 文法ミスの傾向     | 今回の英文から読み取れるミスの癖                       |
| 定量評価           | 文法・語彙・内容を10点満点で採点＋エラー密度・暫定CEFR |
| CEFR評価の根拠     | スコアの判定理由の解説                                 |
| 学習法の提案       | 今のレベルから伸ばすための学習アドバイス               |
| レベルアップ提案   | 一段階上のレベルでの英文サンプル                       |
| 穴埋めクイズカード | ドリルページで出題される復習カードを自動生成           |

**クリアボタン**: 入力内容と結果をリセットします。入力途中の内容はページを移動しても保持されます。

---

### ドリル（弱点克服）

過去の添削結果を使って反復練習するページ。2つのモードがあります。

| モード           | 内容                                                                                          |
| ---------------- | ------------------------------------------------------------------------------------------------ |
| 穴埋めクイズ     | 日付を選び、その日の添削で生成された穴埋めカードを出題する。ヒントを表示すると4択形式に切り替わる |
| 穴あきタイピング | 日付・文を選択し、CEFR一段階上の書き直し文を、マスクを段階的に減らしながらタイピング練習する      |

正解状況は問題ごとに記録され、連続正解数が一定以上になると習熟済みとして出題されにくくなります。

---

### 履歴

過去の添削セッションを確認・管理するページ。

**表示・絞り込み**

- 月表示カレンダーに、セッションのある日がドット表示される
- カレンダーの日付をタップして特定日のセッションに絞り込める
- キーワード検索で本文・ミス内容から該当セッションを検索できる

**セッションの展開**
セッション行をタップすると添削結果の詳細が展開されます。もう一度タップすると閉じます。

**セッションの削除**
各セッションの削除ボタンをタップするか、複数選択して一括削除できます。

**エクスポート（バックアップ）**

1. 画面上部の **エクスポート** ボタンをタップ
2. `history_YYYY-MM-DD.json` という名前でファイルがダウンロードされる

**インポート（復元）**

1. 画面上部の **インポート** ボタンをタップ
2. バックアップした JSON ファイルを選択
3. 既存データと重複しないセッションが追加される（重複は ID で判定）

> インポートする JSON は `CorrectionSession[]` の配列形式である必要があります。

---

### ミス傾向

全添削セッションを集計し、学習の進み具合を可視化するページ。

**スコア推移グラフ**
総合・文法・語彙・内容の4系列で、セッションを重ねるごとのスコア推移を折れ線で表示します。凡例をクリックすると該当系列を強調表示できます。

**CEFR推移グラフ**
セッションごとの暫定CEFR評価の推移を表示します。

**カテゴリ別ミス頻度**
横棒グラフで各カテゴリのミス件数を相対的に表示。右端に件数が表示されます。

**頻出ミスリスト（上位20件）**
同じ誤りを何度も犯している表現を件数の多い順に表示。

| 列         | 内容                   |
| ---------- | ---------------------- |
| 元の表現   | ミスした英語表現       |
| 正しい表現 | 正解の英語表現         |
| カテゴリ   | ミスの種類             |
| 件数       | 同じミスを何回犯したか |

セッションを追加するほど分析精度が上がります。

---

### 実績

添削・ドリルの取り組み状況に応じてバッジが解除されるページ。

- 実績はカテゴリごとに一覧表示され、未解除の項目には進捗バー（あと何回で解除されるか）が表示される
- 「パーフェクト」は穴埋めクイズやレベルアップドリルで1回のセッションを全問正解した回数、
  「連続正解」はドリルでの連続正解数、「継続日数」は練習・ドリルに取り組んだ日数を指す
  （用語の詳細は [glossary.md](glossary.md) を参照）
- 解除済みの実績は達成日とともに表示される

### 設定

アカウント、APIキー、モデル、テーマを管理するページ。編集内容を保存せずに離れようとすると確認ダイアログが表示されます。

#### アカウント

Google アカウントでログイン / ログアウトできます。ログインすると添削履歴が Firebase 経由でクラウドと自動同期され、別端末でも同じ履歴を利用できます。

#### API キー・モデル設定

| 項目            | 説明                                                                                                                                |
| --------------- | ----------------------------------------------------------------------------------------------------------------------------------- |
| Gemini API キー | Google AI Studio で取得したキー。目アイコンで表示/非表示切り替え可                                                                  |
| モデル優先順位  | 使用する Gemini モデルの候補をドラッグ&ドロップで並び替え可能。先頭のモデルから順に試し、失敗すると次の候補へ自動フォールバックする |

#### フィードバック項目

添削では以下のフィードバックが常に含まれます（個別の ON/OFF はありません）。

| 項目                       | 説明                                                   |
| -------------------------- | ------------------------------------------------------ |
| 文法・語法のミスの指摘     | 誤りと正しい表現・理由を解説                           |
| 自然な表現の提案           | ネイティブらしい言い回しを提案                         |
| 文法ミスの傾向分析         | ミスの癖と対策を解説                                   |
| 定量評価（スコア＋CEFR）   | 文法・語彙・内容を10点満点で採点＋エラー密度・暫定CEFR |
| レベルアップした表現の提案 | 一段階上の CEFR レベルの英文サンプル                   |
| 復習カードの自動生成       | 穴埋め＋4択の復習カードを生成（ドリルで出題）          |

#### テーマ

**ライト / ダーク** を切り替えます。設定を保存すると次回起動時も維持されます。

#### バージョン情報

現在のアプリバージョンとリリース日が表示されます。「リリースノートを見る」を開くと、過去バージョン
すべての変更内容（新機能・不具合修正）を確認できます。未読の新バージョンがある場合は、アプリ起動時に
自動でリリースノートのモーダルが表示されます。

#### GitHubリポジトリ

本アプリのソースコードを公開している GitHub リポジトリへのリンクが表示されます。

#### プロンプトプレビュー

添削時に送信される実際のプロンプトを確認できます。

#### テキスト → JSON 変換ツール

過去に書いた英文（添削前の原文）を履歴にインポートするための変換ツール。

1. 日付を選択する
2. 英文を入力する（**＋ 追加** で複数行入力可）
3. **JSON 生成** ボタンをタップ
4. 生成された JSON をコピーして、履歴ページからインポートする

> `corrected`（添削結果）と `mistakes`（ミスリスト）は空になります。添削結果のない記録として履歴に追加されます。

## よくある質問

**Q. API キーを入力したのに添削できない**
A. キーが正しいか確認してください。Google AI Studio でキーが有効になっているかも確認してください。

**Q. 添削結果が遅い**
A. Gemini API のレスポンス時間は文章の長さやモデルによって変動します。設定ページでモデル優先順位を変更すると速くなる場合があります。

**Q. 履歴が消えた**
A. ログインしていない場合、履歴はブラウザの LocalStorage にのみ保存されています。ブラウザのサイトデータを削除すると消去されるため、定期的なエクスポートでのバックアップを推奨します。Google アカウントでログインしていた場合は、クラウド側の履歴からも復元される可能性があります。

**Q. 複数デバイスで同期したい**
A. 設定ページから Google アカウントでログインすると、添削履歴が自動的にクラウド（Firebase）と同期され、他の端末でも同じアカウントでログインするだけで履歴を共有できます。ログインしたくない場合は、従来どおり一方の端末でエクスポートし、もう一方でインポートすることでも履歴を移行できます。
